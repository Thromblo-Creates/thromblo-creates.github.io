Big one: **Idle MMO is now AwayWard**. Same heart, new name, and a *lot* more game
around it.

When this started it was a tidy little combat loop you could leave running in a
Discord channel. Since then it has grown into a proper world, and the name finally
matches the vibe: your hero adventures on while you are away.

## What is new

- **A living world**: factions with shared research, realm-wide world bosses, a
  player market, and housing you can furnish.
- **Deeper progression**: gacha summons with a real pity system, collectible pets
  that level from duplicates, and prestige / rebirth for permanent power.
- **More to do idle**: gathering and crafting run alongside combat, so you are
  always making progress on something.

## And a new home

This site is now the hub for everything I make under **Thromblo Creates**: AwayWard,
the Godot game it grew from, Sage & Pestle, and the desktop tools. There is a dark
theme now too (toggle it bottom-right), and it is hosted on GitHub Pages so it loads
fast from a CDN near you.

Want to jump in? [Add AwayWard to your server](/bots#awayward) or come say hi in the
[Discord](https://discord.gg/aH6K9jhTRb).
